# Privacy Policy

**TechWolf AI-First Toolkit** is a collection of open-source skills and plugins that run entirely within your local Claude Code or Codex environment.

## Data Collection

This toolkit does **not** collect, store, transmit, or process any personal data. There are no analytics, telemetry, tracking, or network requests made by the toolkit itself.

## How It Works

All skills execute locally through your Claude Code or Codex session. Any API calls made during execution are handled by Claude Code or Codex directly, subject to their respective privacy policies:

- [Anthropic Privacy Policy](https://www.anthropic.com/privacy)
- [OpenAI Privacy Policy](https://openai.com/privacy)

## Third-Party Services

The content-studio plugin includes a local Next.js web application that runs on localhost. It does not communicate with external servers.

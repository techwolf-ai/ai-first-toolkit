# Brand Kit Plugin

Official TechWolf brand assets for AI-generated outputs.

## Skills

| Command | What it does |
|---------|--------------|
| `/techwolf-logo` | Provides official logo files in multiple variants |

## Usage

Use the logo skill whenever any output (web app, slide deck, document, PDF) needs a TechWolf logo. The skill bundles all official variants and includes guidance on which to use where.
